# DotArt :)
Simple dot art

```
Usage: DotArt.py [options]
Options:
  -h, --help            show this help message and exit
  -i IMAGE, --image=IMAGE
                        input image
  -o OUTPUTFILE, --output=OUTPUTFILE
                        output file
  -c CONSTRAINTS, --constraints=CONSTRAINTS
                        constraints on count of output symbols
```
